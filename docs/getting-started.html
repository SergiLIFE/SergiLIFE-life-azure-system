Troubleshooting

Common Issues

EEG Signal Quality Problems:
  • Check electrode impedance (should be < 5kŒ©)
  • Ensure proper electrode placement using 10-20 system
  • Minimize electrical interference
  • Use conductive gel for better electrode contact
