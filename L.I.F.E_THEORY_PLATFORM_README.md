## 🚨 Troubleshooting

### Common Issues

#### Platform Won't Load
```cmd

dir life_theory_platform.html

# Try different browser
```

#### Server Mode Issues
```cmd
# Check Python installation
python --version

# Install dependencies
pip install -r requirements.txt
```
