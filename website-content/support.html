4. Troubleshooting

4.1 Common Issues

EEG Signal Quality Issues
  • Check electrode placement and impedance (<5kŒ©)
  • Verify sampling rate matches configuration
  • Ensure minimal electrical interference
  • Calibrate device before session start
API Connection Problems
  • Verify API key validity and permissions
  • Check network connectivity and firewall settings
  • Ensure correct endpoint URLs and HTTPS usage
  • Monitor rate limiting (1000 requests/minute default)
High Latency Issues
  • Optimize network connection quality
  • Use nearest Azure region endpoint
  • Reduce EEG data batch size
  • Enable compression for data transmission

4.2 Performance Optimization

  • Data Preprocessing: Apply bandpass filters (0.5-50 Hz)
  • Artifact Removal: Enable automatic eye blink and muscle artifact detection
  • Batch Processing: Send data in 250ms windows for optimal throughput
  • Caching: Use session caching for repeated API calls
