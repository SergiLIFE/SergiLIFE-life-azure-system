TROUBLESHOOTING
===============

Problem: Terminal 1 shows error after "python life_backend_server.py"
Solution: 
  - Close Terminal 1
  - Go back to STEP 2 in Terminal 1
  - Make sure folder path is exactly:
    c:\Users\Sergio Paya Borrull\OneDrive\Documents\GitHub\.vscode\New folder\SergiLIFE-life-azure-system\SergiLIFE-life-azure-system
  - Try again

Problem: "python: command not found"
Solution:
  - Python is not installed or not in PATH
  - Install Python from python.org
  - Make sure to check "Add Python to PATH" during installation
  - Restart computer
  - Try again

Problem: "No module named flask"
Solution:
  - Go back to STEP 3
  - Make sure pip install completed successfully
  - Try: pip install --upgrade flask flask-cors numpy
  - Wait for completion

Problem: Port 5000 already in use
Solution:
  - Another app is using port 5000
  - Close other apps or:
  - Change line in backend: app.run(host='0.0.0.0', port=5001)
  - Then use: http://localhost:5001

Problem: Port 8080 already in use
Solution:
  - In Terminal 2, instead of: python -m http.server 8080
  - Use: python -m http.server 9000
  - Then open browser to: http://localhost:9000/LIFE_AI_PLATFORM_REAL.html
