# CloudUp Website Builder - Tab Linking Instructions

## If You're Using CloudUp Website Builder Platform

### Step 1: Access CloudUp Editor
1. Log into your CloudUp account
2. Open your website editor/dashboard
3. Navigate to **Menu** or **Navigation** settings

### Step 2: Edit Tab/Menu Item
1. Find the tab you want to link to L.I.F.E. Platform
2. Click **Edit** or **Settings** for that tab
3. Look for **Link** or **URL** field

### Step 3: Configure the Link
**Link Type**: External URL / Custom Link
**URL**: `https://lifecoach-121.com`
**Target**: Open in new window/tab
**Title/Text**: "L.I.F.E. Platform" (or your preferred name)

### Step 4: Save and Publish
1. **Save** the tab configuration
2. **Publish** your website changes
3. **Test** the link to ensure it works
